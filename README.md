# Python Study Together — Данил + Олег

Цель: через 3–4 месяца оба пишем реальные проекты и кидаем в резюме.

## График
- Созвоны: среда 20:00 и воскресенье 18:00 (Discord)
- Ежедневно: минимум 30–60 мин + 1 коммит
- Пропуск созвона без предупреждения = 200 руб на пиццу :)

## План на первые 4–5 недель
- Неделя 1 → Переменные, if/else, циклы
- Неделя 2 → Списки, строки, функции
- Неделя 3 → Словари, множества, файлы, ООП базовый
- Неделя 4 → ООП глубже + рефакторинг
→ С недели 5 — совместные проекты

## Как сдаём домашки
1. Создаёшь ветку `hw1-task1`
2. Пишешь код в папке week1/
3. git push → открываешь Pull Request
4. Я ревьюшу и мержу

## Инструменты
- VS Code + Live Share
- Python 3.12
- Git + GitHub
- Discord


## Основные команды в GIT для Олега
git pull                     # подтянуть свежие изменения
git checkout -b имя-ветки    # создать и перейти в новую ветку
git add .                    # добавить все изменения
git commit -m "сообщение"    # закоммитить
git push origin имя-ветки    # запушить ветку
